# 📚 Bookea

**Bookea** es una plataforma web que conecta a estudiantes y lectores para **comprar, vender o intercambiar libros usados**, principalmente académicos y literarios.

El objetivo es ofrecer un espacio **seguro, accesible y especializado**, fomentando una comunidad lectora activa.

---

## 🚀 Demo

👉 https://bookea.onrender.com/ 

---

## 🎯 Problema que resuelve

- Dificultad para acceder a libros a bajo costo  
- Falta de plataformas especializadas  
- Riesgos de fraude en marketplaces generalistas  
- Desorganización en la búsqueda de libros  

---

## 💡 Solución

Bookea ofrece:

- 📖 Publicación de libros con información detallada  
- 🔍 Búsqueda mediante filtros avanzados  
- ⭐ Wishlist con notificaciones  
- 👥 Validación de usuarios  
- 💬 Contacto entre usuarios (mensajería externa)  
- 🌐 Interfaz responsive

## ⚙️ Funcionalidades principales

- Registro de usuarios  
- Publicación de libros  
- Búsqueda por:
  - título  
  - autor  
  - género  
  - precio  
  - estado  
- Wishlist con alertas  
- Interacción entre usuarios

## 🚧 Alcances y limitaciones

### Incluye
- MVP funcional  
- Plataforma web responsive  
- Funcionalidades básicas de marketplace  

### No incluye
- Pagos dentro de la aplicación  
- Envíos integrados  
- App mobile nativa  
- Sistema de reseñas completo  
- Recomendaciones inteligentes
